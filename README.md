**vue ssr 官方文档见连接 [here](https://ssr.vuejs.org).**

## 构建步骤

**需要 Node.js 7+ （作者使用的版本为：node.js 8.2.1  npm 5.3.0）**

``` bash
# install dependencies
npm install

# serve in dev mode, with hot reload at localhost:80
npm run dev

# build for production
npm run build

# serve in production mode
npm start
```
